Usage¶

Installation¶

  1. To use module, first install requirements.txt:

(.venv) docs$ pip install -r requirements.txt

  1. Create your own reverse proxy to receive webhooks directly in the app in ‘localhost’ and port 8013

For test, I use Ngrok

  1. Create .env file

Note

inside file write API_KEY=<YOUR_API_KEY>

Don’t forget set your address in invoice param: "webHookUrl": "your_address"

If you have 2 and more shops, use param: "reference": "shop_number"

For listening all webhook, you can use handle_webhook() function

webhook_prod.handle_webhook()[source]¶

Handle incoming webhooks, verify their authenticity, and print the response.

This function retrieves the raw data from the incoming webhook request and the ‘X-Sign’ from the headers. It then verifies the webhook using the ‘verify_webhook’ function. If the webhook is verified, it prints “verified” and the JSON response from the request. If the webhook is not verified, it prints “not verified”.

Returns

A dictionary with a single key-value pair, where the key is ‘ok’ and the value is True.

Return type

dict
